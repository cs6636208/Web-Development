6. Flex Grow

CSS
.item {
    flex-grow: 0 | 1 | 2 | 3;
}
HTML
<div class="container">
    <div class="item grow-1">1</div>
    <div class="item grow-2">2</div>
    <div class="item grow-3">3</div>
</div>

7. Flex Shrink

}
HTML
<div class="container">
    <div class="item shrink-0">0</div>

</div>

</div>
